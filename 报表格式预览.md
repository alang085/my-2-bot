# 报表格式预览

## 新报表格式说明

报表已重新设计，更加清晰易懂，主要改进：

1. **结构更清晰**：按逻辑分组（当前状态 → 收入明细 → 支出明细 → 资金总结 → 账户余额）
2. **流动资金放在最后**：作为当日进出账总结，更符合阅读习惯
3. **自动计算合计**：显示收入合计、支出合计，以及净流量计算
4. **格式化数字**：使用千分位分隔符，更易读

---

## 报表格式示例

```
📊 全局报表
📅 今日 (2025-11-28)
生成时间: 2025-11-28 15:30

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 【当前状态】
  有效订单: 50 笔
  有效金额: 1,250,000.00

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📈 【收入明细】
  新客户: 5 笔 | 125,000.00
  老客户: 3 笔 | 75,000.00
  利息收入: 15,000.00
  完成订单: 2 笔 | 50,000.00
  违约完成: 1 笔 | 25,000.00
  ────────────────────────
  收入合计: 290,000.00

📉 【支出明细】
  违约订单: 1 笔 | 30,000.00
  公司开销: 5,000.00
  其他开销: 2,000.00
  ────────────────────────
  支出合计: 37,000.00

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💰 【资金总结】
  流动资金: 253,000.00
  （收入 290,000.00 - 支出 37,000.00 = 253,000.00）

💵 【账户余额】
  现金余额: 1,503,000.00
```

---

## 格式特点

### 1. 标题区域
- 报表类型（全局/归属ID）
- 周期信息（今日/本月/区间）
- 生成时间

### 2. 当前状态
- 有效订单数和金额
- 反映当前业务状态

### 3. 收入明细
- 新客户订单
- 老客户订单
- 利息收入
- 完成订单
- 违约完成订单
- **收入合计**（自动计算）

### 4. 支出明细
- 违约订单
- 公司开销
- 其他开销
- **支出合计**（自动计算）

### 5. 资金总结（放在最后）
- **流动资金**：当日进出账总结
- 计算公式：收入 - 支出 = 净流量
- 清晰展示资金流动情况

### 6. 账户余额
- 当前现金余额
- 反映账户最终状态

---

## 改进点

1. ✅ **流动资金放在最后**：作为当日进出账总结，更符合阅读习惯
2. ✅ **自动计算合计**：收入合计、支出合计自动计算
3. ✅ **清晰的公式展示**：流动资金 = 收入 - 支出
4. ✅ **数字格式化**：使用千分位分隔符（1,250,000.00）
5. ✅ **逻辑分组**：按收入、支出、资金总结分组
6. ✅ **只显示有数据的项**：如果某项为0，则不显示

---

## 使用说明

报表格式已自动应用到：
- `/report` - 今日报表
- `/report <归属ID>` - 归属报表
- 月报查询
- 日期区间查询

所有报表都使用统一的新格式，确保一致性和可读性。
